   command : C:\Program Files\nodejs\node.exe
   args    : un seul argument = chemin ABSOLU vers index.mjs (même dossier que node_modules)

4) Redémarrer le serveur MCP. Clé publique déjà sur PA (install_pa_ssh_key.ps1).
