AVANT TOUT
  - Fichier ".env" (ODOO_URL, ODOO_DB, ODOO_USER, ODOO_PASSWORD)
  - Une fois : INSTALLER_PLAYWRIGHT.cmd puis CONNEXION_ODOO_UNE_FOIS.cmd

CONFIGURATION UNE FOIS (base + rapport + analytique fixes)
  1) Copiez le fichier :
       debug_odoo_defaults.example.json   -->   debug_odoo_defaults.json
     (dans le meme dossier odoo-pythonanywhere)

  2) Ouvrez debug_odoo_defaults.json avec le Bloc-notes.

  3) Collez UNE FOIS l'URL complete du rapport Odoo dans "report_url"
     (barre d'adresse du navigateur quand le rapport est affiche).

  Les champs base_url et analytic_name sont deja remplis pour Ripaille / Aliments PP.
  "dates_ytd": true = periode comme la toolbox (1er jan. -> aujourd'hui).

A CHAQUE FOIS QUE VOUS VOULEZ UN FICHIER debug_pl_bundle.json
  Double-clic :  CAPTURE_DEBUG_RIPAILLE.cmd
  (ou CAPTURE_ET_ENVOYER.cmd / CAPTURE_ET_ENVOYER.ps1 si debug_odoo_defaults.json existe)

  Puis envoyez debug_pl_bundle.json dans le chat Cursor.

Ne commitez pas : debug_odoo_defaults.json, odoo_browser_state.json, debug_pl_bundle.json
