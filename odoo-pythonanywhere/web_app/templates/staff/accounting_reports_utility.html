Même fichier que les autres utilitaires staff. Après enregistrement, la page se recharge avec cette base présélectionnée.

Descriptif & mode d’emploi

But. Cet utilitaire permet d’ajouter une base au registre sans quitter cet écran, de vérifier la connexion, de lister et filtrer les rapports comptables (account.report), d’en personnaliser une copie automatique du rapport choisi (logique Senedoo : notes « custom » retirées sur les feuilles, regroupement par compte, hiérarchie des groupes de comptes, « tout déplier » désactivé) — le rapport d’origine n’est pas modifié, ou d’en supprimer une instance (si Odoo et vos droits l’autorisent).

  • Nouvelle base : dépliez « Ajouter une base Odoo au registre » ci-dessus → enregistrement puis rechargement avec la base sélectionnée.
  • Choisissez le client / base (regroupement par libellé ; précision prod/test dans la liste), puis cliquez sur Actualiser : le voyant indique si l’API répond.
  • Les noms affichés sont en français (contexte Odoo fr_FR, surcharge possible via variable TOOLBOX_ODOO_LANG). Le filtre porte sur le libellé dans cette langue.
  • Saisissez un filtre (ex. compte, SYSCOHADA, résultat) pour réduire la liste.
  • Personnaliser : indiquez l’id du rapport source (standard ou existant), tapez OUI — une copie est créée dans Odoo puis seule cette copie est personnalisée ; la barre de progression s’affiche pendant l’envoi du formulaire (navigation classique).
  • Supprimer : recopiez exactement la phrase demandée (SUPPRIMER-<id>) — irréversible si Odoo accepte la suppression.
