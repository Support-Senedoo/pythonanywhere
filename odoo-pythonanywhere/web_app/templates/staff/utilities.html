Personnalisation des rapports comptables

Parcours distincts selon l’objectif. Les écrans qui listent des rapports affichent les account.report de la base choisie.

  • Compte de résultat personnalisé — copie d’un rapport P&L + personnalisation Senedoo détail comptes / SYSCOHADA. Renommage et suppression depuis la liste.
  • Balance OHADA (6 colonnes) — crée ou supprime un rapport Balance OHADA (six colonnes débit/crédit initiaux et finaux).
  • Compte de résultat analytique + Wizards Odoo — tableau CR analytique par axe (réalisé / budget / %, export Excel) ; sur la même page après sélection d’un client : Wizard CPC Budget Analytique et Tableau de Bord Manager (installation en un clic dans Odoo).

📈 Tableau de Bord Manager (Senedoo)

Installe un wizard natif dans Odoo SaaS : KPIs Ventes, Achats, Facturation, Trésorerie, Stock, Fabrication — comparaison N vs N-1 automatique, charte graphique Senedoo, sélection société et axe analytique. Accessible dans Odoo : Comptabilité > Rapports > Tableau de Bord Manager (Senedoo).
