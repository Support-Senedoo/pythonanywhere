Sonde des bases Odoo par compte

But & principe

Deux modes (non enregistrés côté appli ; login et mot de passe masqués à l’écran) : (1) laissez l’URL vide : connexion au portail Odoo.com (page « Mes bases »), liste des instances *.odoo.com liées au compte, puis test XML-RPC sur chacune. (2) URL d’une instance : db.list() sur cette instance si le service db existe, puis authentification sur chaque base listée. Indications si accès OK : sociétés, module base, utilisateur.

  • Mode portail : dépend du HTML de odoo.com — en cas d’échec, vérifiez langue (variables d’environnement TOOLBOX_ODOO_PORTAL_LANG, TOOLBOX_ODOO_PORTAL_ORIGIN).
  • Mode URL : si db.list() est absent (souvent Odoo SaaS 19+), la liste reste vide — utilisez alors le mode sans URL.
  • Maximum {{ max_probe }} bases testées par requête (sécurité / performance).
  • Ne pas utiliser sur une machine partagée sans confiance : identifiants transmis à odoo.com et aux instances.

Auteur : Senedoo · Version : {{ util_version }} · Date : {{ util_date }}

Liste des bases (portail Mes bases) : renseignez login + mot de passe, puis « Analyser ». L’URL plus bas est facultative — laissez-la vide pour ce mode.
