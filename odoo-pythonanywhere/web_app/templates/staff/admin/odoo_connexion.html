Captcha odoo.com, cookie portail, PythonAnywhere

Le portail www.odoo.com déclenche souvent un captcha depuis le serveur de la toolbox (IP datacenter). Après captcha dans un navigateur : script scripts/capture_odoo_portal_cookie_playwright.py, puis TOOLBOX_ODOO_PORTAL_COOKIE ou …_FILE (voir toolbox-env-exemple.txt), ou collez le Cookie de la requête document vers …/my/databases. Le mot de passe / clé API reste nécessaire pour tester l’XML-RPC sur chaque base.

Alternative : URL d’instance si db.list() est disponible (souvent absent sur Odoo SaaS récent).
