Mode d’emploi — budget financier, analytique et CPC (Odoo)

Deux leviers dans Odoo. (1) Copie P&L pilotage (optionnel) : si vous connaissez l’id d’un account.report source, la toolbox en duplique une copie avec filtres analytique et budgets — pour parcourir les rapports dans un tableau, utiliser l’utilitaire P&L personnalisé. Réinstallez le wizard depuis la toolbox après une mise à jour pour appliquer les correctifs d’expressions « % » (évite la division par zéro si budget = 0).
