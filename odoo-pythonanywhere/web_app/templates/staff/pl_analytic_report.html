Descriptif — compte de résultat analytique (API)

But. Calculer un tableau réalisé / budget / % par compte général pour un compte analytique (projet) et une période, via l’API Odoo (account.move.line, analytic_distribution, crossovered.budget.lines). Aucun rapport account.report n’est créé dans la base — contrairement à l’utilitaire « P&L analytique / budget » qui duplique un rapport Odoo.

Période : les dates début / fin délimitent l’intervalle (écritures et budgets selon chevauchement). Rapport comptable : liste filtrable comme sur « P&L personnalisé » — si vous en choisissez un, le tableau est restreint aux comptes généraux couverts par les expressions « account_codes » des lignes feuilles (même logique que la personnalisation SYSCOHADA). Sinon, tous les comptes avec mouvement analytique pour le projet sont listés.

Compte analytique : id numérique (account.analytic.account). Les montants réalisés suivent analytic_distribution (prorata des %).
